# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-03

### Added
- Initial release of FastAPI Enum-Dict
- CLI commands: init, create, list, show, update, delete
- Enum management (file-based storage)
- Dict management (database-based storage)
- Automatic type detection with Unicode support
- 21 Jinja2 templates for code generation
- Complete documentation (README, installation guide, quick start)
- Unit tests with 100% pass rate
- SQLAlchemy 2.0 and Pydantic integration

### Fixed
- Detector Chinese character detection bug
- PowerShell file encoding issues in tests

### Documentation
- Complete Chinese README
- Installation guide
- Quick start guide
- Quick reference card
- Testing guide
- API documentation (auto-generated)
